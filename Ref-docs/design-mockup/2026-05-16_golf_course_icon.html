골프장 아이콘 시안 — 3종 비교

Custom SF Symbol 또는 SwiftUI Shape으로 제작. 사용자가 1개 선택 후 SwiftUI Asset Catalog에 추가.

Option 1 — 깃발 + 그린 + 홀
잔디 언덕 위 깃발 + 홀(컵). 골프장의 명확한 표현. 가장 직관적.
Option 2 — 깃발 + 원형 그린 (위에서 본 시점)
평면 그린 + 펄럭이는 깃발. Material 'golf_course'에 가장 가까운 표현.
Option 3 — 깃발만 (미니멀)
단순한 깃대 + 깃발. SF Symbols flag.fill과 비슷한 톤. 가장 미니멀.
구현 방식: 선택된 옵션은 (a) SwiftUI Shape으로 직접 그리거나 (b) SVG → SF Symbols Mac 앱으로 export 후 Asset Catalog .symbolset 추가.
다크 자동: stroke/fill 색을 .tint로 받게 하면 라이트(#2E7D32)/다크(#66BB6A) 자동.
