B. 가로 미니 테이블 (스펙 §4 정합)
기존 spec 31-VIEWER §4 그대로. 9홀 가로 행 + par 행 + 플레이어 행 + 합계 sticky. par 셀은 색상 없이 plain, par-diff만 마커. 모바일 가로 480px 안에 fits.

결정 요청

A안 — 동반자가 1명일 때 가장 깨끗. 각 홀에 홀#·Par·스코어가 묶여있어 시각적 그룹 명확. 다인 라운드면 player별 grid 반복 = 세로 길어짐.

B안 — spec §4 정합. 다인 라운드(2~4명) 시 각 플레이어가 한 행씩 추가되어 비교 쉬움. 1인 라운드엔 빈공간 많음.

공통 개선: Hero 합계 카드 (총 타수 + par diff 크게) + par-diff 색상 매트릭스 정확 적용 (현재 빨강계 1타도 다 eagle 처리되던 버그 수정).
