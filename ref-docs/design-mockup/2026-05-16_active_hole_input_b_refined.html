설정 — 벌타 기본값 추가
OB / 해저드 / 컨시드 각각 1–5 사이 stepper로 사용자 설정.
default: OB +2 / 해저드 +1 / 컨시드 +1 (현재 코드 값과 동일).
홀 입력 모드의 벌타 버튼 라벨은 이 값을 실시간 반영 ("OB +2" → "OB +3" 변경 시 즉시 라벨 갱신).

동작 시나리오

  1. 새 라운드 시작 → 기본 "보기 모드" (현재 스코어보드 UI 그대로) 또는 "입력 모드" 진입 (사용자 마지막 선택 기억)
  2. 입력 모드: 가운데 큰 숫자는 활성 플레이어의 현재 홀 타수. 탭 액션 없음(표시 전용)
  3. 왼쪽 [−] 탭 → −1 (최소 0) / 오른쪽 [+] 탭 → +1 (최대 par × 2)
  4. OB 탭 → 설정에서 로드한 값만큼 타수 자동 증가 (default +2)
  5. 해저드 / 컨시드도 동일하게 설정값만큼 추가
  6. 플레이어 전환은 상단 4인 탭 한 번. 각 탭에 현재 홀 점수 함께 표시 ("나 / 4")
  7. ← / → 으로 홀 이동. 자동 다음홀 이동은 기본 안 함 (의도치 않은 진행 방지)
  8. Par 4 badge 탭 → menu(3/4/5)로 즉시 변경 (보기 모드와 동일)
  9. 모드 토글(상단)로 보기 ↔ 입력 즉시 전환. 두 모드는 동일 데이터 sharing
