### 2. 생활_골프장.csv

- **출처**: data.go.kr/data/15045080/fileData.do (행안부 LOCALDATA)
- **인코딩**: UTF-8 (BOM 없음)
- **총 행수**: 649개 (영업/취소/휴업 모두 포함 — 필터링 필요)
- **컬럼 (31)**: 개방자치단체코드 / 관리번호 / 인허가일자 / 인허가취소일자 / **영업상태명** ⭐ / 폐업일자 / 휴업시작일자 / 휴업종료일자 / 소재지우편번호 / 도로명우편번호 / **사업장명** / 데이터갱신구분 / 건축물동수 / 건축물연면적 / 공사립구분명 / 데이터갱신시점 / **도로명주소** / 문화체육업종명 / 법인명 / 보험가입여부코드 / **상세영업상태명** ⭐ / 상세영업상태코드 / 세부업종명 / 영업상태코드 / 전화번호 / **좌표정보(X)** ⭐ / **좌표정보(Y)** ⭐ / 지도자수 / **지번주소** / 회원모집총인원 / 최종수정시점

**좌표계**: **EPSG:5174 (TM, 중부원점)** → WGS84(EPSG:4326) 변환 필요. 변환 라이브러리: `pyproj` 또는 `proj`

```python
from pyproj import Transformer
transformer = Transformer.from_crs("EPSG:5174", "EPSG:4326", always_xy=True)
lng, lat = transformer.transform(x, y)
```

**필터링 필요**: 영업상태명 = "영업/정상" 또는 "휴업/정지"만 채택. "취소/말소/만료/정지/중지", "폐업" 등은 제외.
