## 2. 동성 동반자 누락 수정

- [ ] **같은 성이 2명 이상인 실제 스코어카드**로 불러오기 (예: `이**`, `김**`, `이**`, `정**`)
- [ ] 4명이 **전원** 들어오는지 (기존엔 3명)
- [ ] 각자의 점수가 **뒤섞이지 않았는지** (기존엔 살아남은 `이**`에 세 번째 사람 점수가 들어감)
- [ ] 본인(owner)이 **정확히 1명만** 지정되는지 ← 파생 결함으로 함께 고친 부분.
      같은 성 2명이 모두 owner가 될 뻔했음
- [ ] 전반/후반 점수가 각자에게 올바로 붙는지

## 3. 홈 불러오기 CTA (UI)

- [ ] **위계** — hero(라운드 시작)와 경쟁하지 않는지. 라운드 시작이 여전히 주행동으로 읽히는지
- [ ] **라이트/다크 모두** 확인. 특히 다크에서 카드 배경과 `accentGreen` 15% 아이콘 박스 대비
- [ ] **빈 상태**(라운드 0건) — 텍스트 버튼이 주 CTA(초록 알약)를 침범하지 않는지
- [ ] **작은 기기(SE 크기)** — 빈 상태에서 GPS 안내까지 세로가 넘치지 않는지
- [ ] **큰 글씨 설정** — CTA 부연 문구가 `lineLimit(1)`이라 잘리는지 확인

## 참고 — 미배포 상태

- **Worker og:image 미배포**: 코드는 푸시됐으나 `wrangler deploy --env production` 전이다.
  배포 전까지 **카톡 통계 공유 미리보기 이미지는 나오지 않는다**(앱이 `ogImage`를 보내도
  현재 배포본은 그 필드를 몰라 무시 — 무해).

  **PIN 건 공유는 이미지가 안 뜨는지**(의도된 동작).
