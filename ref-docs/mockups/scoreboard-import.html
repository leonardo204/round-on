⑤ 저장 직전 검증
핵심 인터랙션 메모
  • 스코어 값은 PAR 대비 가감(스코어카드 표기 그대로). 파3 홀에 「1」이면 내부 4타로 환산해 저장. 사용자는 카드 표기와 1:1 비교만 확인하면 됨.
  • 두 섹션(힐/크리크)을 탭 전환 없이 위·아래로 동시 표시. 하단 패널은 세로 스크롤.
  • PAR 행은 읽기 전용(코스 기준). +/− 조정은 본인 + 동반자 점수 셀에만 적용.
  • 셀 탭 → 그 셀이 active(파란 외곽) → 하단 +/− stepper로 점수 조정. 동시에 이미지 viewer가 해당 셀 영역으로 줌.
  • 의심 셀(빨강)·null 셀은 자동으로 stepper 포커스. 정상 셀은 탭으로 선택.
  • 스코어 그리드는 9홀 + TOTAL 고정 10열 — 가로 스크롤 없음.
  • 합계 자동 검증은 하지 않음. 저장 후 ‘이번 라운드’에서 자유 편집.
  • 원본 이미지는 저장하지 않음 (사진 저장 폐기 정책 준수).
